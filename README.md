FX1 Digital Hub
MiniPay-powered Creator Economy on Celo

⸻

🌍 Overview

FX1 Creator Hub is a mobile-first Web3 application built on the Celo blockchain, designed to empower creators, artists, and NFT fashion designers to earn seamlessly through decentralized payments.

By integrating MiniPay, FX1 Creator Hub enables users to send and receive instant payments in cUSD, making creator monetization accessible, fast, and borderless.

This project sits at the intersection of Art, NFT Fashion, and Blockchain, aligning with Celo’s mission of financial inclusion and global accessibility.

⸻

✨ Features

* 💳 MiniPay Integration – Seamless mobile payments using cUSD
* 🎨 Creator Tipping System – Support artists directly onchain
* 📱 Mobile-First Experience – Optimized for MiniPay users
* 🔗 Wallet Connection – Easy onboarding with Web3 wallets
* 🧾 Proof of Creation – Onchain record for creative works

⸻

🛠️ Tech Stack

* Frontend: React / Vite
* Blockchain: Celo
* Smart Contracts: Solidity
* Wallet Integration: WalletConnect / MiniPay
* Deployment: Celo Alfajores (Testnet) / Mainnet

⸻

🚀 How It Works

1. Connect your wallet (MiniPay supported)
2. Explore creators and their works
3. Send tips instantly using cUSD
4. Creators receive payments directly onchain
